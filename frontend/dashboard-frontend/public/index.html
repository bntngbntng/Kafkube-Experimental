Database Schema

Mahasiswa Table Schema

Column Type Constraints
nim VARCHAR(10) PRIMARY KEY
nama VARCHAR(100) NOT NULL
jurusan VARCHAR(50) NOT NULL
created_timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP

Example Data

NIM Nama Jurusan
200210001 John Doe Teknik Informatika
200210002 Jane Smith Telkom
200110045 Robert Johnson Teknik Elektro
200310033 Emily Chen MMB
200120077 Ahmad Fauzi GT
The mahasiswa table stores all student records. The nim (student ID) is the primary key with format YYYYJJNNN, where:
  • YYYY: Year of enrollment
  • JJ: Department code
  • NNN: Sequential number
